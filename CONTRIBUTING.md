# Contributing to AgentFlow
We're looking for contributions in secure execution, distributed systems, and observability domains. Good first issues are labeled `good-first-issue`.

## Development Workflow:
1. Run `make dev-env` to start local development containers
2. Use `cargo clippy` for linting
3. All code must have FFI-safe abstractions for memory-critical components

## Design Reviews
Propose architecture changes via RFCs in docs/rfcs directory.
